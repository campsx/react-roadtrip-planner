## Custom commands

| Command | File | Description |
|---------|------|-------------|
| `/review` | `.claude/commands/review.md` | Full professional project review — generates `CLAUDE_ACTION/REVIEW.md` with architecture, quality, perf, security scores. |
| `/new-page` | `.claude/commands/new-page.md` | Generates a new page with its complete feature-based structure. |
| `/new-api` | `.claude/commands/new-api.md` | Generates a new API module (one file per function + `index.ts`). |
| `/refacto` | `.claude/commands/refacto.md` | Produces `CLAUDE_ACTION/REFACTO.md` with the action plan, asks for confirmation, executes, then updates the rules. |
| `/new-feature` | `.claude/commands/new-feature.md` | Scaffolds a new feature slice — generates `CLAUDE_ACTION/FEATURE.md`, asks for confirmation, then creates all files. |

> All files generated by commands are written to `CLAUDE_ACTION/` at the project root.
